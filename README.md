# mongodb with express
Mongodb with mongoose package

These repo will work as a starting point of working with mongodb using mongoose package and express web framework

# Install dependencies 
after cloning or downloading repo, issue the following command in the project directory: 
npm install 

# Run 
nodemon server.js 

or 

npm run dev 

**Installation links:**
